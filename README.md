MONGO-GeoHashing
================

Introduction 
-------------------------------------------------------------
A complete project in which a MySql db is used as source of Points data, read, geohashed, and saved in Mongodb.

Requirements
-------------------------------------------------------------
    PHP 5.2 or greater
    CodeIgniter 2.1.0 to 3.0-dev
	Mongodb 2.4.8 or greater
	
Manual Installation
-------------------------------------------------------------

1) Download Package
   
2) Move into target directories

-------------------------------------------------------------
